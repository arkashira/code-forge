# breakeven.md
## Unit Economics
### Cost per Active User
Based on the infrastructure requirements for the code-forge platform, we estimate the following costs per active user:
* Compute: $0.05 per user per month (assuming 10% utilization of a $5/month server)
* Storage: $0.01 per user per month (assuming 100MB storage per user at $0.10/GB-month)
* Bandwidth: $0.005 per user per month (assuming 100MB bandwidth per user at $0.05/GB)
Total cost per active user: $0.065 per user per month

## Pricing Tiers
We propose the following pricing tiers for the code-forge platform:
### Tier 1: Hobbyist
* Price: $9/month
* Features:
	+ Limited to 100 code generations per month
	+ Access to basic skill library
	+ Community support only
### Tier 2: Professional
* Price: $29/month
* Features:
	+ Unlimited code generations
	+ Access to premium skill library
	+ Priority support
### Tier 3: Enterprise
* Price: $99/month
* Features:
	+ Unlimited code generations
	+ Access to custom skill library
	+ Dedicated support and onboarding

## Customer Acquisition Cost (CAC) Range
Based on industry benchmarks, we estimate the CAC range for the code-forge platform to be between $15 and $30 per user.

## Lifetime Value (LTV) Estimate
Assuming an average user retention rate of 75% per month, we estimate the LTV for each tier as follows:
* Tier 1: $9/month \* 12 months \* 0.75 (retention rate) = $81 per user
* Tier 2: $29/month \* 12 months \* 0.75 (retention rate) = $261 per user
* Tier 3: $99/month \* 12 months \* 0.75 (retention rate) = $891 per user

## Break-even Users Count
To calculate the break-even point, we need to consider the CAC and LTV for each tier. Assuming a CAC of $20 per user, we get:
* Tier 1: $81 (LTV) - $20 (CAC) = $61 profit per user
* Tier 2: $261 (LTV) - $20 (CAC) = $241 profit per user
* Tier 3: $891 (LTV) - $20 (CAC) = $871 profit per user
To reach break-even, we need to cover the costs of development, infrastructure, and marketing. Assuming a total monthly cost of $5,000, we get:
* Tier 1: $5,000 (monthly cost) / $61 (profit per user) = 82 users
* Tier 2: $5,000 (monthly cost) / $241 (profit per user) = 21 users
* Tier 3: $5,000 (monthly cost) / $871 (profit per user) = 6 users

## Path to $10K MRR
To reach $10,000 MRR, we can aim for a combination of users across the three tiers. Assuming a mix of 20% Tier 1, 60% Tier 2, and 20% Tier 3 users, we get:
* Tier 1: 20% \* $9/month = $1.80 per user per month
* Tier 2: 60% \* $29/month = $17.40 per user per month
* Tier 3: 20% \* $99/month = $19.80 per user per month
Average revenue per user (ARPU): ($1.80 + $17.40 + $19.80) / 3 = $12.67 per user per month
To reach $10,000 MRR, we need: $10,000 / $12.67 (ARPU) = 790 users
Assuming the same user distribution, we get:
* Tier 1: 20% \* 790 users = 158 users
* Tier 2: 60% \* 790 users = 474 users
* Tier 3: 20% \* 790 users = 158 users
This combination of users would generate approximately $10,000 MRR.
