# Library Metrics

This project provides a simple library for viewing detailed metrics for each library.

## Usage

1. Create a `Library` object with the library name and metrics.
2. Call the `get_metrics` method to retrieve the library metrics.
3. Use the `get_placeholder` method to retrieve a placeholder for missing metrics.
4. Use the `get_ci_pipeline_link` method to retrieve the CI pipeline link.

## Testing

Run the tests using `pytest`.
