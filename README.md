# 🚀 Code Forge

**Power AI coding agents with high-quality code.** Code Forge is a skill library platform for AI coding agents to generate high-quality code for real-world software tasks.

---

## Why Code Forge?

* **🔥 Fast Development**: Code Forge enables AI coding agents to generate high-quality code quickly, reducing development time and increasing productivity.
* **🤖 AI-Powered**: Code Forge is designed to work seamlessly with AI coding agents, providing a library of skills and knowledge to power their code generation capabilities.
* **📚 Comprehensive Library**: Code Forge offers a comprehensive library of code snippets, APIs, and tools to help AI coding agents generate high-quality code for a wide range of software tasks.
* **🔒 Secure**: Code Forge is built with security in mind, providing a secure and reliable platform for AI coding agents to generate code.
* **📈 Scalable**: Code Forge is designed to scale with your needs, providing a flexible and adaptable platform for AI coding agents to generate code.

## Getting Started

### Install Dependencies

```bash
poetry install
```

### Run Tests

```bash
pytest
```

### Run Code Forge

```bash
python src/library_api.py
```

---

## Deploy

### Deploy to Production

```bash
poetry run python src/library_api.py
```
