# Code Forge
A VS Code plugin for skill discovery and insertion.

## Features
* Search for skills by keyword, tag, and version
* Insert code snippets with one click
* Display CI badge and documentation link for each skill

## Installation
1. Clone the repository
2. Run `poetry install`
3. Open VS Code and install the plugin

## Usage
1. Open the Command Palette in VS Code
2. Type "Code Forge: Search Skills" and select the command
3. Enter your search query and select a skill
4. Click "Insert" to insert the code snippet
